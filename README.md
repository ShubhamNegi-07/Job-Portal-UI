# 💼 Job Board UI

A modern and responsive **Job Board UI** built with **React**. It features clean job cards, company branding, job tags, salary information, and an intuitive interface for exploring tech job opportunities.

# ⚡ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/ShubhamNegi-07/job-board-ui.git
```

### 2. Navigate into the project

```bash
cd job-board-ui
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```
